    Class VineNull

    VineNull represents a schema type that only accepts null values. Typically used inside union types to explicitly allow null as a valid value.

    Example

    const schema = vine.union([
      vine.string(),
      vine.null()
    ])

    const result = await vine.validate({
      schema,
      data: null
    })
