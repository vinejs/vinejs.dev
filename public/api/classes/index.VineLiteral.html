    Class VineLiteral<Value>

    VineLiteral represents a schema type that matches an exact literal value. It validates that the input exactly matches the specified literal value, supporting string, number, boolean, and null literals.

    Example

    const schema = vine.literal('admin')

    const result = await vine.validate({
      schema,
      data: 'admin'
    })
